Implement CausalPy Interrupted Time Series for campaign-level incrementality scoring and Hill function saturation curve modeling, both via TDD. Purpose: STAT-02 (campaign-level incrementality), STAT-03 (confidence intervals), and STAT-06 (saturation curves) are the core analytical capabilities. Incrementality is the product's primary differentiator — telling marketers which campaigns actually cause additional revenue versus what would have happened anyway.

**GREEN — Implement (packages/analysis/models/saturation.py):**

`hill_saturation_percent(spend_series: np.ndarray, revenue_series: np.ndarray, recent_days: int = 30) -> dict`

- Compute saturation_pct = hill(current_spend) / alpha
- Return dict with saturation_percent, hill_alpha, hill_mu, hill_gamma, status
