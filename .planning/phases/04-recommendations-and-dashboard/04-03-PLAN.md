**A. Create TanStack Query hooks:**

`useKpis(dateRange, comparisonRange?)`:
- Fetches `/api/dashboard/kpis?tenantId=X&from=...&to=...` (+comparison params if enabled)
- queryKey: ['kpis', from, to, compareFrom, compareTo]
- staleTime: 5 * 60 * 1000 (5 minutes)

`useRecommendations(tenantId)`:
- Fetches `/api/recommendations?tenantId=X`
- queryKey: ['recommendations', tenantId]
- staleTime: 5 * 60 * 1000

`useCampaigns(dateRange, platform?, level?)`:
- Fetches `/api/dashboard/campaigns?tenantId=X&from=...&to=...&platform=...&level=...`
- queryKey: ['campaigns', from, to, platform, level]
- staleTime: 5 * 60 * 1000
