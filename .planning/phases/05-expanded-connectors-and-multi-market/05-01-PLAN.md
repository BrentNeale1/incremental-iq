    Create `packages/ingestion/src/market-detection/` directory with three files.

    **google-ads.ts â€” `detectGoogleAdsMarkets()`**
    Uses the two-query GAQL approach from RESEARCH.md Pattern 4:
    1. Query 1: SELECT campaign.id, campaign_criterion.location.geo_target_constant FROM campaign_criterion WHERE type='LOCATION' AND negative=FALSE
    2. Query 2: SELECT geo_target_constant.resource_name, geo_target_constant.country_code FROM geo_target_constant WHERE resource_name IN (...)
    3. Build Map<campaignExternalId, Set<countryCode>> from results
    4. Use `country_code` field from ALL geo constants regardless of target_type (RESEARCH.md Pitfall 4 â€” cities/regions have correct country_code too)
    5. Import the existing GoogleAdsConnector pattern from `connectors/google-ads.ts` for GAQL query execution (use the same `customer.query()` pattern)
    6. Accept parameters: accessToken, customerId, loginCustomerId, tenantId
    7. Return: Array<{ campaignExternalId: string, countryCodes: string[] }>

    **meta.ts â€” `detectMetaMarkets()`**
    Uses RESEARCH.md Pattern 5:
    1. Use facebook-nodejs-business-sdk (same require() pattern as connectors/meta.ts) to fetch ad sets with targeting field
    2. Extract `targeting.geo_locations.countries` array from each ad set (ISO 2-letter codes)
    3. Aggregate countries up to campaign level (union of all ad set countries per campaign)
    4. Ad sets with no geo_locations.countries = campaign goes to Global/Unassigned (empty array)
    5. Accept parameters: accessToken, adAccountId, tenantId
    6. Return: Array<{ campaignExternalId: string, countryCodes: string[] }>

    **index.ts â€” `detectMarketsForTenant()`**
    Orchestrator that:
    1. Reads all integrations for tenantId (platform='google_ads' and platform='meta')
    2. Decrypts tokens using existing `decryptToken()` from packages/ingestion
    3. Calls detectGoogleAdsMarkets for each Google Ads integration
    4. Calls detectMetaMarkets for each Meta integration
    5. Aggregates all country codes across all integrations
    6. For each unique country code: upsert into `markets` table with displayName from `Intl.DisplayNames` (RESEARCH.md Don't Hand-Roll), campaignCount = number of campaigns targeting that country, isConfirmed=false
    7. For each campaign: upsert into `campaign_markets` table with source='auto_detected', resolve campaignExternalId to internal UUID via campaigns table lookup
    8. All DB operations use withTenant() for RLS context
    9. Returns the created/updated markets array
